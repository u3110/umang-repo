About Project

Project Topic Implementation and Analysis of various Visual Cryptographic techniques.

Overview

  • The primary objective of this project is to implement various Cryptographic techniques for transmission of images securely.
  • Implementation and analysis of various visual Cryptographic techniques on the basis of different parameters.
  • Provide secure transmission of colored images.

Techniques Used

  1. Image Encryption based on Bit-plane Decomposition and Random Scrambling.
  2. Visual Cryptography (VC) using Zigzag Scan Approach.
  3. Color Image Cryptography Scheme Based on Visual Cryptography(YCbCr).
  4. Visual Cryptography For Image Depend on RSA and Elgamal Algorithms.

Languages & Platforms

  • Backend:

Conclusion Performance analysis of various Visual Cryptographic technique.
